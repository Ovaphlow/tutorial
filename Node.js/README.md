# Node.js

## npm

### 设置 registry 使用国内镜像

```shell
npm config set registry https://registry.npm.taobao.org
```

## Yarn

### 安装

```shell
npm install yarn --global
```

### 设置 registry 使用国内镜像

```bash
yarn config set registry https://registry.npm.taobao.org
```

## gRPC

```shell
npm install --grpc_node_binary_host_mirror=https://npm.taobao.org/mirrors/
```
